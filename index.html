Guide

1.Usage:

1.1 Simple Use
      
      //init  
      var myguide = $.guide()  
      //add step
      myguide.add('.selector', 'step1 tip-message').add('.selector2', 'this is step2')   
      //start the guide
      myguide.start()
Go!
1.2 Auto Guide
      
      //parse the 3rd para a number 
      var myguide = $.guide().add('.selector', 'step1 auto', 500)
                             .add('.selector2', 'auto step2', 500)
                             .start();
Go!

2.Options:

      
      //init
      var options= {
          //默认是否调用targe.click()
          bundle: false,
          //every guide before callback
          onGuideBefore: noop,
          //every guide after callback
          onGuideAfter: noop,
          //all guide done callback
          onGuideDone: noop,
          //start callback
          onGuideStart: noop,
          //tip位置
          tip_margin: 75,
          //tip class
          tip_class: "guide_tip",
      }

      var myguide = $.guide(options)
paramdefaultdescribe
bundlefalse不传递bundle参数时的默认bundle参数
tip_margin75提示信息和target的距离
tip_classguide_tip提示信息的class
onGuideStartnullbefore start callback
onGuideBeforenullevery guide before callback
onGuideAfternullevery guide after callback
onGuideDonenullafter all guide done callback
onGuideStartnullbefore start callback

3.Methods:

3.1 add

add one guide step

        add:function(target, message, bundle, callback){}
paramdescribe
target 被引导元素的选择器
message 显示的引导信息
bundle

可以设置两种值，来达到不同效果:

1.设置为数字时，表示这个引导使用定时器，经过bundle毫秒之后，就会继续下一步

2.设置为boolean值时，true，表示点击引导的时候也会触发targe的click事件，不填或者false表示不触发
callback 引导结束之后的回调函数
3.2 start

Start the guide

        start:function(){}
